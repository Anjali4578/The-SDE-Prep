

### ➡️ 1.Reverse a String
```cpp
#include <algorithm>
//User function Template for C++

string reverseWord(string str){
    int left=0,right=str.length()-1;
    
    while(left<right){
        swap(str[left++],str[right--]);
    }
    // reverse(str.begin(), str.end());
    return str;
    
  //Your code here
}
```
